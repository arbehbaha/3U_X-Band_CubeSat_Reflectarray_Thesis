# 3U_X-Band_CubeSat_Reflectarray_Thesis
This repository contains the work, Python scripts, HFSS Projects, and KiCad Projects used in the development of the thesis: A PROOF-OF-CONCEPT DESIGN OF A HIGH-GAIN DEPLOYABLE REFLECTARRAY ANTENNA FOR A 3U CUBESAT

# Repository Name

Brief description of what this repository contains and what the project is for.

## Repository Structure

This repository is organized into the following folders:

### `Chamber_Code`

Contains code related to chamber control, measurement automation, data acquisition, or experimental setup workflows.

### `HFSS_Projects`

Contains Ansys HFSS project files, simulation setups, electromagnetic models, and related design files.

### `Images`

Contains images used for documentation, reports, figures, diagrams, plots, or README visuals.

### `Kicad_Projects`

Contains KiCad PCB design files, schematics, layouts, footprints, symbols, and related hardware design resources.

### `Scripts`

Contains utility scripts for processing data, automating workflows, generating plots, running analyses, or supporting simulations.

### `Unit_Cell_Raw_Simulation_Data`

Contains raw simulation data for unit cell studies, including exported results, parameter sweeps, and unprocessed simulation outputs.
