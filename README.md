# 3U_X-Band_CubeSat_Reflectarray_Thesis
This repository contains the work, Python scripts, HFSS Projects, and KiCad Projects used in the development of the thesis: A PROOF-OF-CONCEPT DESIGN OF A HIGH-GAIN DEPLOYABLE REFLECTARRAY ANTENNA FOR A 3U CUBESAT

## Repository Structure

This repository is organized into the following folders:

### `Chamber_Code`

Contains code related to the development of the antenna testing range used to measure the reflectarray performance. The most updated version of the main test code is 'Full_Chamber_Sweep_v3.py'. This code performs positioner control, data acquisition, and measurement automation.

### `HFSS_Projects`

Contains Ansys HFSS project files and simulation results.

### `Images`

Contains media and images used in the final report.

### `Kicad_Projects`

Contains KiCad projects used to create the reflectarray layout designs.

### `Scripts`

Contains general scripts used to perform computations such as phase distributions across the reflectarray surface, and to generate DXF of the reflectarray geometry.

### `Unit_Cell_Raw_Simulation_Data`

Contains raw simulation data for unit cell studies, including exported results, parameter sweeps, and unprocessed simulation outputs.
